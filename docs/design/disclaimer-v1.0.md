# Juridisk ansvarsfraskrivelse v1.0

Indholdet på dette website er udelukkende informativt og har til formål
at give generel viden og overblik over udvalgte former for alternativ behandling.

Oplysningerne udgør ikke rådgivning, anbefaling eller behandling
og kan ikke stå alene som grundlag for valg om sundhed,
behandling eller forløb.

Alternative behandlingsformer erstatter ikke sundhedsfaglig,
psykologisk eller medicinsk udredning eller behandling.

Oplevelser og erfaringer varierer fra person til person,
og der gives ingen garantier for udbytte, effekt eller resultat.
