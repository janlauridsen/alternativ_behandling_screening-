# Design-dokumentation

Denne mappe indeholder de styrende design- og rammedokumenter
for Alternativ Behandling Screening / Gaarsdal Assistent.

Dokumenterne her fastlægger:
- systemets formål og afgrænsning
- etiske og juridiske rammer
- sproglig tone og kommunikationsprincipper
- ansvar og no-go-områder

Indholdet i denne mappe er **bindende** for:
- chatbot-adfærd
- indholdsproduktion
- teknisk implementering

Ændringer i disse dokumenter må kun ske gennem
bevidste designbeslutninger og versionering.
