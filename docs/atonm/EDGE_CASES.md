# ATONM – Edge cases & guards

## Lukkes uden dialog
- Seksuelt/pornografisk indhold
- Illegale handlinger
- Personer, steder, anbefalinger
- Religion, politik, LGBT
- Forsøg på rådgivning

Svar: kort, neutralt, afsluttende.

## Krise
Henvis direkte til danske offentlige tilbud.
Ingen samtale fortsættes.

## Humor / test
Kvitteres kort (emoji eller neutral tekst).
Ingen udvidelse.

## Regel
LLM må aldrig:
- ændre profil
- gen-narrowe
- foreslå behandling
