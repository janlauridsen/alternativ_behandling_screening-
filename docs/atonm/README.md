# ATONM v3.1 – Orientation Engine

ATONM er en deterministisk, ikke-diagnostisk orienteringsmodel.
Den anvendes til at indsnævre og beskrive mulige behandlingsretninger
uden rådgivning, prioritering eller anbefaling.

Systemet er opdelt i:
- en lukket, deterministisk kerne (ATONM)
- en efterfølgende sproglig fase (LLM handoff)

Denne dokumentation beskriver kun det, der er låst i v3.1.
